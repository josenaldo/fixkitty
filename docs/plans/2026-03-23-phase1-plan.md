## Phase 1.3 — Infrastructure: Ubuntu 24

- [ ] Create `ProcessBuilderCommandRunner` implementing `CommandRunner`
  - [ ] Execute command via `ProcessBuilder`
  - [ ] Capture stdout and stderr
  - [ ] Enforce timeout via `Process.waitFor(timeout, TimeUnit.SECONDS)`
  - [ ] Return `StepStatus.TIMEOUT` on timeout
  - [ ] Return `StepStatus.FAILED` for non-zero exit codes
  - [ ] Javadoc on class and all public methods
- [ ] Create `SudoPrivilegeManager` implementing `PrivilegeManager`
  - [ ] Prepend `sudo` to the command array
  - [ ] Javadoc
- [ ] Create `PkexecPrivilegeManager` implementing `PrivilegeManager`
  - [ ] Wrap command with `pkexec`
  - [ ] Javadoc
- [ ] Create `Ubuntu24EnvironmentDetector` implementing `EnvironmentDetector`
  - [ ] Read distro from `/etc/os-release`
  - [ ] Read desktop from `$XDG_CURRENT_DESKTOP`
  - [ ] Detect PipeWire via `systemctl --user is-active pipewire`
  - [ ] Detect graphical session via `$DISPLAY` / `$WAYLAND_DISPLAY`
  - [ ] Javadoc
- [ ] Create `Ubuntu24ActionCatalog` implementing `ActionCatalog`
  - [ ] FIX_AUDIO: `systemctl --user restart pipewire` (requiresPrivilege=false), `systemctl --user restart wireplumber` (requiresPrivilege=false)
  - [ ] FIX_BLUETOOTH: `systemctl restart bluetooth` (requiresPrivilege=true, onFailure=ABORT)
  - [ ] FIX_NETWORK: `systemctl restart NetworkManager` (requiresPrivilege=true, onFailure=ABORT)
  - [ ] FIX_GNOME_SHELL: `killall gnome-shell` (requiresPrivilege=false, onFailure=ABORT)
  - [ ] FIX_ALL: audio → bluetooth → network → gnome shell
  - [ ] CHECK_ENVIRONMENT: empty step list
  - [ ] Javadoc
- [ ] Register all implementations in `AppModule`
- [ ] Integration test for `ProcessBuilderCommandRunner` with a safe real command (e.g., `echo hello`)

**Deliverable:** Full Ubuntu 24 infrastructure; end-to-end flow testable from use case to OS.
