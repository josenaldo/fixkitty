# AI Journal — FixKitty

Registro transparente do uso de IA no desenvolvimento do FixKitty.

## Formato de cada entrada

Cada sessão gera um arquivo `YYYY-MM-DD-NNN-titulo.md` com:
- **Data e contexto** da sessão
- **Skills usadas** (brainstorming, writing-plans, subagent-driven-development, code-reviewer, etc.)
- **Decisões importantes** tomadas
- **O que funcionou** bem
- **O que não funcionou** ou poderia melhorar
- **Métricas** (tarefas, testes, commits)

## Propósito

Servir como fonte fidedigna para retrospectivas, posts de blog e aprendizado sobre uso de IA no desenvolvimento de software.
